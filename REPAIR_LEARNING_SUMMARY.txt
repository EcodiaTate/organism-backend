================================================================================
  KNOWN ISSUES (From Audit Prompt 2)
================================================================================

🐛 CRITICAL (Must fix before production):

1. Fix Type Matching Broken
   • Problem: Uses statement substring, not explicit fix_type field
   • Impact: Proposals will be incorrectly flagged as missing fix
   • Example: "add_route_handler" won't match proposal text "add route handler"
   • Root cause: Underscore vs space mismatch in matching logic
   • Fix: Extract fix_type metadata explicitly, normalize (underscore ↔ space)

⚠️ HIGH (Should fix):

2. Metadata Missing
   • Problem: endpoint, fix_type, incident_class stored only in narrative
   • Impact: Can't trace hypothesis back to source repair
   • Suggestion: Add metadata dict to Hypothesis with structured fields

3. Hypothesis Persistence
   • Problem: Repair hypotheses stored only in memory
   • Impact: Hypotheses lost on service restart
   • Mitigation: Check if HypothesisEngine has Neo4j persistence layer
